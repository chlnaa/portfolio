# Portfolio

[English](./README.md) | [한국어](./README.ko.md) | [日本語](./README.ja.md)

### 📌 Introduction

A high-performance personal portfolio website built with **React 19** and **Vite**.
This project showcases modern frontend development practices, focusing on clean UI/UX and smooth animations.

### 🛠️ Tech Stack

- **Core**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **Icons**: Lucide React, React Icons

### 🚀 Key Features

- **Direct Navigation**: Each project card links directly to its specific GitHub repository for better accessibility.
- **UX Refinement**: Implemented `select-none` on contact links to prevent accidental text highlighting.
- **Multilingual Support**: Fully localized content in English, Korean, and Japanese.
- **Responsive Design**: Optimized layout for all devices (Mobile, Tablet, Desktop).

### 🏃 Getting Started

# Install dependencies

```bash
npm install
```

# Run development server

```bash
npm run dev
```

# Build for production (Type-check included)

```bash
npm run build
```

## Project structure (high level)

- `src/App.tsx`: page composition (hero/about/experience/skills/projects/contact)
- `src/components/`: UI sections and shared components
- `src/context/`: app-level providers (theme, language)
- `src/constants/`: static data used by the UI
