# 🚀 Modern Portfolio 2026

> **High-performance personal portfolio** built with **React 19** and **Vite**, focusing on clean UI/UX and seamless localization.

---

## 🔗 Live Demo

[👉 Visit My Portfolio Website](https://choinaa-portfolio.vercel.app/)

## 🌟 Key Highlights

### 🌍 Multilingual Support (i18n)

- **English, Korean, and Japanese**: Fully localized content to reach a global audience.
- **Context API**: Managed language state globally for consistent UI rendering.

### 🎨 UX & Performance

- **Optimized Interactions**: Implemented `select-none` on contact links and used **Framer Motion** for smooth, non-intrusive animations.
- **Type-Safe Development**: Leveraged **TypeScript** to ensure robust component structures and error-free builds.
- **Modern UI Kit**: Built with **Tailwind CSS** and **shadcn/ui** for a professional and consistent design system.

### 📱 Responsive & Accessible

- **Mobile**: Fully responsive layout optimized for all devices.
- **Direct Accessibility**: project card links directly to its specific GitHub repository for transparent code review.

## 🛠️ Tech Stack

- **Core**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **Icons**: Lucide React, React Icons
- **Deployment**: Vercel

## 🚀 Key Features

- **Direct Navigation**: Each project card links directly to its specific GitHub repository for better accessibility.
- **UX Refinement**: Implemented `select-none` on contact links to prevent accidental text highlighting.
- **Multilingual Support**: Fully localized content in English, Korean, and Japanese.
- **Responsive Design**: Optimized layout for all devices (Mobile, Tablet, Desktop).

## 📂 Project Structure

- `src/App.tsx`: page composition (hero/about/experience/skills/projects/contact)
- `src/components/`: UI sections and shared components
- `src/context/`: App-level providers for **Theme** and **Language**.
- `src/constants/`: static data used by the UI

## 🏃 Getting Started

**Install dependencies**

```bash
npm install
```

**Run development server**

```bash
npm run dev
```

**Build for production**

```bash
npm run build
```
